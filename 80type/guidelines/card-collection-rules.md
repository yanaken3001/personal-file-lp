# カードコレクション＆バイラルループ仕様

## バイラルループ（確定）

```
Aさん診断完了
→ 自分のカード1枚獲得
→ マイフォルダに1枚目追加（残り79枠が空欄で見える）
→ 「友達のカードを集めよう！」CTA
→ シェアリンク生成
→ Bさんがリンクから診断
→ Bさんの結果 ＋ AさんとBさんの相性結果が同時表示
→ Aさんのフォルダに「Bさんのカード」が自動追加
→ Aさんに通知「Bさんのカードを獲得！相性82%⚡最強コンビ」
→ 相性カード画像が自動生成
→ バイラルループ継続
```

## バイラル加速の3つの仕掛け
1. **空欄の79枠が見える** — コンプリート欲を刺激
2. **相性カードは2人の合体カード** — 1枚の画像としてシェアできる
3. **レア演出** — 出現率の低いタイプ獲得時にエフェクト豪華に

## シェアリンクの仕様
- 形式：`https://80cards.jp/invite/[ユーザーID]`
- パラメータ：`?from=[招待者ID]` で招待トラッキング
- 相性表示：`?match=[比較対象ID]` で相性結果ページ直行

# キャラクターデザイン仕様マニュアル

## 共通デザインルール（確定・変更不可）

| ルール | 内容 |
|---|---|
| 頭身比 | 二頭身（chibi two-head-tall proportions） |
| 体表面の質感 | マットなソフト3D（soft matte 3D rendering style） |
| 美術方向性 | Sticky Monster Lab / Pictoplasma / ビニールトイ的 |
| 顔の構造 | 鼻なし。口は小さな線。目が表情の主役 |
| 構図 | 正面向き（front-facing symmetrical view） |
| 背景 | クリーンな白背景 |
| 生成ツール | lovart.ai（Fluxモデル） / 1:1 |

## キャラクター画像の使用ルール
- 画像ファイルは `character/` フォルダに格納（16体すべて完成済み）
- ファイル名形式：`[類型コード] [あだ名].png`（例：`PP チームの太陽.png`）
- 画像の加工・変形・色変更は**絶対に禁止**
- カード内での配置は面積の40-50%を占めるサイズ
- 背景透過が必要な場合は白背景を除去して使用する

## UIデザインへの適用指針
- キャラクターのマット3D質感に合うUI（フラットすぎず、リアルすぎず）
- 各グループのカラーをUI全体に展開（ヘッダー、ボタン、アクセント等）
- キャラクターが映える背景設計（白/ダーク/グラデーション）
- キャラクターを「主役」にするレイアウト（テキストで埋め尽くさない）

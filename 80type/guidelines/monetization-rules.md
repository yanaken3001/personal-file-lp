# マネタイズ＆LINE導線マニュアル

## LINE登録導線（確定）

```
カード10枚達成
→ 「あなたの取扱説明書（トリセツ）が完成しました！」
→ 「LINEで受け取る」ボタン
→ LINE友だち追加
→ トリセツPDF自動配信
→ ナーチャリング開始
```

## セグメント別分岐（確定）

```
LINE登録ユーザー
├─ 20代 × 転職意思あり → 自社転職支援（最高単価）
├─ 20代 × 転職意思なし × スキル不安 → オンラインスクール（アフィリエイト）→ ナーチャリング継続
├─ 20代 × 転職意思なし × 現職満足 → 有料コンテンツ → 長期エンゲージメント
├─ 30代 → 30代向け転職エージェント（アフィリエイト）
├─ 40代以上 → 40代特化型スクール（アフィリエイト）
└─ 学生 → 就活対策サービス（アフィリエイト）→ 卒業後に自社転職支援へ
```

## セグメント情報の収集タイミング

| タイミング | 収集情報 | 方法 |
|---|---|---|
| 診断完了時 | 性格タイプ, 行動タイプ | 自動（診断結果から） |
| LINE登録時 | 年代, 性別 | 登録フォーム |
| LINE初回配信 | 転職意思, 現在の状況 | アンケート形式メッセージ |
| 3日後配信 | スキルへの関心領域 | タップ選択式 |

## マネタイズ上のルール
- 診断自体は完全無料。課金ウォールは絶対に設けない
- LINE登録は「特典の受け取り手段」として自然に誘導する（強制しない）
- セグメント情報収集はユーザー体験を損なわない範囲で行う
- アフィリエイト導線は「あなたのタイプに合ったおすすめ」として提案する（押し売りしない）

## KPI設計

| 指標 | 目標値（初期） |
|---|---|
| 診断完了率 | 70%以上（開始した人のうち） |
| シェア率 | 30%以上（診断完了した人のうち） |
| カード3枚達成率 | 20%以上 |
| カード10枚達成率 | 5%以上 |
| LINE登録率 | 80%以上（10枚達成者のうち） |
| バイラル係数 | 1.5以上（1人が平均1.5人を招待） |

# 80CARDS プロジェクト概要

## サービス名
- **80CARDS**（エイティカーズ）— サービス名。固有名詞。商標登録対象
- **80タイプの性格診断** — サブタイトル。初見の人に1秒で伝える

## プロダクト構造

### 表の顔
80タイプに分かれる性格診断。友達のカードを集めるコレクション体験。

### 裏の顔
属性別に自動分類されたリードを、最適なマネタイズ先にCVさせるリードジェネレーションプラットフォーム。

## ポジショニング
- MBTIと同様に、ウェブ上で直接診断できるサービス
- 個人情報登録やLINE連携は不要で診断可能（従来の仕様からの変更点）
- LINE登録はカード10枚達成時の特典（トリセツPDF）受取で自然に誘導

## 診断仕様
- 全57問（性格軸32問 + 行動軸25問）
- 所要時間：約7分
- 無料・登録不要で診断可能

## 核となる体験
1. **診断** → 自分のタイプがわかる（16タイプ × 5行動 = 80タイプ）
2. **カード獲得** → 自分のカードが1枚もらえる
3. **シェア** → 友達にリンクを送る
4. **相性** → 友達の結果＋2人の相性が同時表示
5. **コレクション** → 友達のカードがフォルダに追加される（残り79枠が見える）
6. **特典アンロック** → 枚数に応じて特典が解放される
7. **LINE登録** → 10枚達成でトリセツPDF、ナーチャリング開始

## 競合との差別化
- MBTI（16Personalities）：16タイプ止まり → 80CARDSは80タイプでより精密
- 動物占い：コレクション要素なし → 80CARDSはカード収集＋相性でバイラル
- エニアグラム：難解 → 80CARDSはキャラクターで親しみやすい

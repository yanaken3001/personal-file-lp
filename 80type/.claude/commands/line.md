# /line — LINE/CRM部を起動

LINE配信・ナーチャリング・セグメント設計に関する指示を処理します。

## 起動エージェント
- **kei**（LINEマーケター）— 配信シナリオ・セグメント分岐・リッチメニュー

## 起動手順
1. `agents/7-line-crm/kei.md` を読み込む
2. `guidelines/monetization-rules.md` を必ず読み込む
3. `guidelines/card-collection-rules.md` と `guidelines/type-system.md` も必要に応じて読み込む
4. テンプレートが必要な場合 `templates/line-scenario.md` を使用
5. Agent toolでサブエージェントとして起動する

## 使用例
```
/line トリセツPDF受け取りのLINE登録フローを設計して
/line 20代×転職意思ありのナーチャリングシナリオを作って
/line リッチメニューのデザイン案を出して
/line ステップ配信の7日間シナリオを設計して
```

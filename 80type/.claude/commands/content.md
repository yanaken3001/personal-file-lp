# /content — コンテンツ制作部を起動

テキスト・コピー・説明文に関する指示を処理します。

## 起動エージェント
- **mio**（ライター）— タイプ説明文・相性テキスト・キャッチコピー
- **tomo**（UXライター）— 画面コピー・ボタン文言・CTA・通知テキスト

## ルーティング判断
- 「説明文」「タイプ解説」「相性テキスト」「キャッチコピー」「トリセツ」→ mio起動
- 「ボタン」「CTA」「通知」「画面テキスト」「UXコピー」「エラーメッセージ」→ tomo起動
- 「トーン」「ブランドボイス」等の複合タスク → 2名を並列起動

## 起動手順
1. `agents/5-content/mio.md` または `agents/5-content/tomo.md` を読み込む
2. `guidelines/brand-guidelines.md` と `guidelines/type-system.md` を必ず読み込む
3. 既存のタイプデータ参照が必要な場合 `index.html` 内のTYPES_80を参照
4. Agent toolでサブエージェントとして起動する

## 使用例
```
/content PP（チームの太陽）の説明文を書いて
/content 相性テキストのテンプレートを作って
/content 診断開始画面のCTAコピーを5案出して
/content カード獲得通知のメッセージを考えて
```

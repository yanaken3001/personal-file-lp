# /design — デザイン部を起動

UIデザイン・カードデザイン・ビジュアル設計に関する指示を処理します。

## 起動エージェント
- **hana**（UIデザイナー）— 画面設計・デザインシステム・レイアウト
- **ren**（カードデザイナー）— カード画像・演出設計・コレクションUI

## ルーティング判断
- 「UI」「画面」「レイアウト」「配色」「フォント」「デザインシステム」→ hana起動
- 「カード」「演出」「エフェクト」「コレクション画面」「カードデザイン」→ ren起動
- 「結果画面」「デザインコンセプト」等の複合タスク → 2名を並列起動

## 起動手順
1. `agents/2-design/hana.md` または `agents/2-design/ren.md` を読み込む
2. `guidelines/brand-guidelines.md` と `guidelines/character-design-rules.md` を必ず読み込む
3. キャラクター画像の参照が必要な場合 `character/` フォルダを参照
4. Agent toolでサブエージェントとして起動する

## 使用例
```
/design 結果画面のUIを設計して
/design カード画像のレイアウトを提案して
/design 80CARDSのデザインコンセプトを考えて
/design コレクションフォルダの空欄デザインを提案して
```
